# SGA en tu Laboratorio

Aplicación React para consultar condiciones de almacenamiento, etiquetado y manejo de residuos de reactivos químicos según el SGA.

## Instrucciones

1. Clona el repositorio:
   git clone https://github.com/tu-usuario/sga-laboratorio-app.git

2. Instala dependencias:
   npm install

3. Ejecuta el proyecto:
   npm run dev

## Construido con
- Vite
- React
- Tailwind CSS
- jsPDF, xlsx, file-saver
